# JavaScript Objects, Date, Math

## 1. Object Cơ Bản

- **Khai báo**: Sử dụng `{ key: value, ... }`.
- **Thêm/Sửa**: `obj.key = value`. Nếu dùng biến làm key, dùng cú pháp ngoặc vuông: `obj[variable] = value`.
- **Lấy value**: `obj.key` hoặc `obj['key']` (nếu sai trả về `undefined`).
- **Xóa key**: `delete obj.key`.
- **Khái niệm**: 
  - **Phương thức (Methods)**: Là các function bên trong object.
  - **Thuộc tính (Properties)**: Các key còn lại mang giá trị thông thường.

## 2. Object Constructor (Hàm tạo)

Dùng để tạo bản thiết kế cho nhiều đối tượng tương tự nhau (nên viết hoa chữ cái đầu).
Có thể khởi tạo qua Declaration Function hoặc Expression Function.

```javascript
function User(firstName, lastName) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.getName = function() {
        return `${this.firstName} ${this.lastName}`;
    };
}

var author = new User('Huy', 'Nguyen');
console.log(author.getName()); // "Huy Nguyen"
```
- Sử dụng `author.constructor` để biết đối tượng được khởi tạo từ constructor nào.

## 3. Object Prototype

Cho phép thêm thuộc tính và phương thức vào constructor **bên ngoài** định nghĩa hàm tạo.
Các thành phần nằm trên `prototype` sẽ được chia sẻ chung cho tất cả các object được tạo từ constructor đó.

```javascript
User.prototype.className = 'F8';
console.log(author.className); // "F8"
```

## 4. Đối tượng Date

Sử dụng `new Date()` để khởi tạo và lấy thời gian hiện tại.
Các phương thức chính lấy thời gian:
- `getFullYear()`: Lấy năm hiện tại.
- `getMonth()`: Lấy tháng hiện tại (Lưu ý: từ `0` đến `11`).
- `getDate()`: Lấy ngày hiện tại trong tháng.
- `getHours()`, `getMinutes()`, `getSeconds()`: Lấy giờ, phút, giây hiện tại.

## 5. Đối tượng Math

Cung cấp các thuộc tính và phương thức toán học tĩnh (không cần `new`):
- **Thuộc tính**: `Math.PI` (Số Pi).
- **Làm tròn**: 
  - `Math.round()`: Làm tròn số thông thường.
  - `Math.ceil()`: Làm tròn lên (trần).
  - `Math.floor()`: Làm tròn xuống (sàn).
- **Khác**:
  - `Math.abs()`: Lấy giá trị tuyệt đối.
  - `Math.random()`: Trả về số thực ngẫu nhiên từ 0 đến dưới 1.
  - `Math.min(a, b, ...)`: Giá trị nhỏ nhất trong các tham số.
  - `Math.max(a, b, ...)`: Giá trị lớn nhất trong các tham số.
