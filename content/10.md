# Các Phương Thức Xử Lý Mảng & Chuỗi Nâng Cao

Trong bài học này, chúng ta sẽ tìm hiểu về các phương thức mạnh mẽ giúp duyệt và thao tác với Mảng (Array) và Chuỗi (String) một cách chuyên nghiệp, thay thế cho các vòng lặp truyền thống (`for`, `while`).

---

### 1. Phương thức `forEach()`
Dùng để duyệt qua từng phần tử của mảng và thực thi một hành động nào đó.
- **Tham số:** `function(currentValue, index, array)`
- **Đặc điểm:** Không tạo ra mảng mới, luôn trả về `undefined`. Không thể dùng `break` để thoát vòng lặp.
- **Ví dụ:**
```javascript
const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach((item, index) => {
    console.log(`${index}: ${item}`);
});
```

### 2. Phương thức `every()`
Kiểm tra xem **tất cả** các phần tử trong mảng có thỏa mãn một điều kiện nhất định hay không.
- **Giá trị trả về:** `true` (nếu tất cả đều đúng), `false` (nếu có ít nhất một cái sai).
- **Cơ chế:** Dừng ngay lập tức (short-circuit) khi gặp phần tử đầu tiên trả về `false`.
- **Ví dụ:**
```javascript
const numbers = [2, 4, 6, 8];
const isAllEven = numbers.every(num => num % 2 === 0); // true
```

### 3. Phương thức `some()`
Kiểm tra xem **có ít nhất một** phần tử trong mảng thỏa mãn điều kiện hay không.
- **Giá trị trả về:** `true` (nếu có 1 cái đúng), `false` (nếu tất cả đều sai).
- **Cơ chế:** Dừng ngay lập tức khi gặp phần tử đầu tiên thỏa mãn điều kiện (`true`).
- **Ví dụ:**
```javascript
const numbers = [1, 3, 5, 8];
const hasEven = numbers.some(num => num % 2 === 0); // true (vì có số 8)
```

### 4. Phương thức `find()`
Tìm và trả về **phần tử đầu tiên** thỏa mãn điều kiện.
- **Giá trị trả về:** Giá trị phần tử đó, hoặc `undefined` nếu không tìm thấy.
- **Cơ chế:** Dừng ngay lập tức khi tìm thấy phần tử thỏa mãn.
- **Ví dụ:**
```javascript
const users = [{id: 1, name: 'An'}, {id: 2, name: 'Bình'}];
const user = users.find(u => u.id === 2); // {id: 2, name: 'Bình'}
```

### 5. Phương thức `filter()`
Lọc ra **tất cả** các phần tử thỏa mãn điều kiện và gom chúng vào một mảng mới.
- **Giá trị trả về:** Mảng con chứa các phần tử thỏa mãn, hoặc mảng rỗng `[]` nếu không có gì.
- **Ví dụ:**
```javascript
const numbers = [1, 2, 3, 4, 5, 6];
const evens = numbers.filter(num => num % 2 === 0); // [2, 4, 6]
```

### 6. Phương thức `map()`
Tạo ra một mảng mới bằng cách **biến đổi** từng phần tử của mảng cũ theo logic cho trước.
- **Giá trị trả về:** Mảng mới có cùng độ dài với mảng cũ.
- **Ví dụ:**
```javascript
const names = ['an', 'binh'];
const upperNames = names.map(name => name.toUpperCase()); // ['AN', 'BINH']
```

### 7. Phương thức `reduce()`
Dùng để "thu gọn" mảng thành một giá trị duy nhất (kết quả tích lũy).
- **Tham số:** `function(accumulator, currentValue, index, array)` và một giá trị khởi tạo `initialValue` (tùy chọn).
- **Lưu ý:**
    - Nếu có `initialValue`: `accumulator` bắt đầu bằng giá trị này, lặp `n` lần.
    - Nếu không có: `accumulator` lấy phần tử đầu tiên, lặp `n-1` lần.
- **Ví dụ:**
```javascript
const prices = [10, 20, 30];
const total = prices.reduce((sum, current) => sum + current, 0); // 60
```

### 8. Phương thức `includes()`
Kiểm tra sự tồn tại của một giá trị bên trong một Mảng hoặc một Chuỗi.
- **Cú pháp:** `target.includes(searchElement, fromIndex)`
- **Ví dụ với Chuỗi:**
```javascript
"Javascript".includes("Java"); // true
```
- **Ví dụ với Mảng:**
```javascript
[1, 2, 3].includes(2); // true
```
