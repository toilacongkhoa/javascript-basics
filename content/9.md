# Vòng lặp & Xử lý Tập hợp

### 1. Vòng lặp For
- **Cơ bản:** `for (khởi_tạo; điều_kiện; bước_nhảy) { ... }`
- *Mẹo:* Có thể bỏ trống các thành phần nhưng phải giữ dấu `;`.

### 2. Vòng lặp `for...in` (Lấy Key/Index)
- **Object:** Lấy tên thuộc tính (key).
- **Array/String:** Lấy chỉ số (index).

### 3. Vòng lặp `for...of` (Lấy Giá trị)
- Dùng cho Array, String, Set... (Iterable).
- **Array:** Lấy giá trị phần tử.
- **String:** Lấy từng ký tự.

### 4. Hỗ trợ Object & Set
- **Object.keys(obj):** Mảng các key.
- **Object.values(obj):** Mảng các value.
- **new Set(array):** Tạo mảng chỉ chứa các phần tử duy nhất.
