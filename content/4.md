# JavaScript String (Chuỗi trong JavaScript)

## 1. Khai báo chuỗi
- **Nguyên thủy (Primitive):** `var a = 'khoa';`
- **Tạo bằng Object:** `var a = new String('khoa');` (Lưu ý: `typeof` sẽ trả về `object`)

## 2. Các thao tác cơ bản
- **Kiểm tra kiểu dữ liệu:** Dùng toán tử `typeof`.
- **Ký tự Escape (\\):** Để in ra dấu nháy hoặc dấu sẹc, dùng `\'`, `\"`, hoặc `\\`.
- **Lấy độ dài chuỗi:** Dùng thuộc tính `.length`.
- **Template String (ES6):** Dùng backtick (`` ` ``) để truyền trực tiếp biến vào chuỗi.
  ```javascript
  var a = 'world';
  console.log(`hello ${a}`);
  ```

## 3. Các phương thức xử lý chuỗi thường dùng
- **Lấy ký tự tại vị trí Index:**
  - `charAt(index)`: Trả về ký tự tại `index` (nếu sai index trả về chuỗi rỗng `''`).
  - `[index]`: Sử dụng ngoặc vuông giống Mảng (nếu sai index trả về `undefined`).
- **`indexOf(str, [start])`**: Tìm vị trí đầu tiên chuỗi `str` xuất hiện. Trả về `-1` nếu không có.
- **`lastIndexOf(str, [start])`**: Tìm vị trí cuối cùng chuỗi `str` xuất hiện. Trả về `-1` nếu không có.
- **`search(str)`**: Tương tự `indexOf` (hỗ trợ Regex) nhưng không có tham số bắt đầu.
- **`slice(start, [end])`**: Cắt chuỗi từ `start` đền `end` (không bao gồm `end`). Hỗ trợ index là số âm.
- **`replace(old, new)`**: Thay thế chuỗi `old` bằng `new`. Mặc định chỉ thay thế kết quả đầu tiên tìm thấy.
- **`toUpperCase()` / `toLowerCase()`**: Chuyển toàn bộ thành chữ in hoa / in thường.
- **`trim()`**: Loại bỏ khoảng trắng thừa ở hai đầu mút của chuỗi.
- **`split(separator)`**: Tách chuỗi thành một Mảng (Array) dựa trên `separator`. Truyền `''` để tách từng chữ cái.
