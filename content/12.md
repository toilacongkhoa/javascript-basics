# Bài 12: DOM (Document Object Model)

## DOM là gì?

Khi trình duyệt tải một trang HTML, nó không xử lý HTML như là văn bản thuần túy. Thay vào đó, trình duyệt **chuyển đổi (parse) toàn bộ chuỗi HTML thành các đối tượng JavaScript** — gọi là **DOM (Document Object Model)**.

DOM có **3 loại node** chính:

| Loại node   | Mô tả                                          | Ví dụ                    |
|-------------|------------------------------------------------|--------------------------|
| **Element** | Tương đương với mỗi thẻ HTML                   | `<h1>`, `<div>`, `<p>`   |
| **Text**    | Nội dung văn bản bên trong thẻ                 | `"Hello World"`          |
| **Attribute** | Các thuộc tính của thẻ HTML                  | `id="title"`, `class="box"` |

> **Lưu ý:** Mỗi thẻ HTML tương đương với **một element node** trong DOM.

---

## Đối tượng `document`

`document` là **đối tượng DOM đại diện cho toàn bộ trang web**. Đây là điểm khởi đầu để truy cập và thao tác với bất kỳ phần tử nào trên trang.

```js
console.log(document); // toàn bộ DOM của trang
console.log(document.title); // tiêu đề trang
```

---

## Các cách lấy Element

### 1. Lấy theo `id` — `getElementById`

```js
const element = document.getElementById('my-id');
```

- Trả về **trực tiếp một element** (hoặc `null` nếu không tìm thấy).
- Nhanh nhất vì `id` là duy nhất.

---

### 2. Lấy theo `class` — `getElementsByClassName`

```js
const elements = document.getElementsByClassName('my-class');
```

- Trả về một **HTMLCollection** (tập hợp các element).
- **Cập nhật động** (live): nếu DOM thay đổi, collection này tự cập nhật theo.

---

### 3. Lấy theo tên thẻ — `getElementsByTagName`

```js
const paragraphs = document.getElementsByTagName('p');
```

- Trả về một **HTMLCollection** gồm tất cả các thẻ khớp.

---

### 4. Lấy theo CSS Selector — `querySelector` & `querySelectorAll`

```js
// Chọn phần tử đầu tiên khớp với selector
const first = document.querySelector('.container h1');

// Chọn tất cả phần tử khớp với selector
const all = document.querySelectorAll('.item');
```

| Phương thức         | Trả về                          | Ghi chú                          |
|---------------------|---------------------------------|----------------------------------|
| `querySelector`     | Một element (hoặc `null`)       | Dùng khi cần chọn 1 phần tử     |
| `querySelectorAll`  | **NodeList** (giống array)      | Dùng khi cần chọn nhiều phần tử |

> **Lưu ý về NodeList:** NodeList **trông giống array** nhưng **không có các phương thức của array** như `.map()`, `.filter()`. Dùng `Array.from(nodeList)` để chuyển sang array thật.

```js
const items = document.querySelectorAll('.item');
const itemArray = Array.from(items);
itemArray.map(item => console.log(item.textContent));
```

---

### 5. Lấy qua HTML Collection có sẵn

Trình duyệt cung cấp sẵn các collection đặc biệt:

```js
document.forms        // tất cả thẻ <form>
document.images       // tất cả thẻ <img>
document.links        // tất cả thẻ <a> có href
document.anchors      // tất cả thẻ <a> có name
document.scripts      // tất cả thẻ <script>
document.styleSheets  // tất cả stylesheet
document.title        // tiêu đề trang (string)
```

---

### 6. Gọi tiếp các phương thức get trên element con

Sau khi lấy được một element, ta có thể **tiếp tục gọi các phương thức get** trên element đó để tìm kiếm trong phạm vi nhỏ hơn:

```js
const form = document.getElementById('login-form');
const inputs = form.getElementsByTagName('input'); // chỉ tìm trong form
const submit = form.querySelector('[type="submit"]');
```

---

## `document.write()`

```js
document.write('<h1>Hello</h1>');
```

> ⚠️ **Cảnh báo:** Nếu dùng `document.write()` **sau khi DOM đã tải xong**, nó sẽ **xóa toàn bộ nội dung trang** và ghi đè bằng nội dung mới. Không nên dùng trong thực tế.

---

## Thao tác với Attribute

### Gán giá trị cho attribute có sẵn

```js
const heading = document.getElementById('title');
heading.title = 'Đây là tiêu đề'; // thuộc tính title
heading.id = 'new-id';            // thuộc tính id
```

> ⚠️ **Đặc biệt với `class`:** Không dùng `.class` (vì `class` là từ khóa JavaScript), phải dùng `.className`:

```js
heading.className = 'highlight'; // gán class
```

---

### Thêm attribute tùy chỉnh — `setAttribute`

```js
element.setAttribute('data-role', 'admin');
element.setAttribute('aria-label', 'Nút đóng');
```

Dùng để thêm **bất kỳ attribute nào**, kể cả attribute tự định nghĩa (`data-*`).

---

### Lấy giá trị attribute — `getAttribute`

```js
const role = element.getAttribute('data-role'); // 'admin'
const label = element.getAttribute('aria-label'); // 'Nút đóng'
```

---

### Lấy toàn bộ danh sách attribute — `attributes`

```js
const allAttrs = element.attributes; // NamedNodeMap
```

- Trả về một **NamedNodeMap** chứa tất cả attribute của element.
- **Chỉ hợp lệ** với các attribute có trong thẻ HTML gốc.
- Attribute tự thêm bằng JavaScript (không khai báo trong HTML) có thể **không xuất hiện** ở đây.

---

## Lấy và thay đổi nội dung text

### `innerText` vs `textContent`

```js
const el = document.getElementById('box');
console.log(el.innerText);    // text hiển thị trên trình duyệt
console.log(el.textContent);  // toàn bộ text, kể cả ẩn
```

| Thuộc tính    | Lấy gì?                                                             |
|---------------|---------------------------------------------------------------------|
| `innerText`   | Chỉ lấy text **đang được hiển thị** trên trình duyệt               |
| `textContent` | Lấy **toàn bộ text**, kể cả: ẩn (`display:none`), khoảng trắng, xuống dòng, nội dung trong `<script>` |

**Ví dụ HTML:**
```html
<div id="box">
  Hello <span style="display:none">World</span>
</div>
```

```js
document.getElementById('box').innerText;    // "Hello"
document.getElementById('box').textContent;  // "Hello World"
```

**Thay đổi nội dung text:**
```js
const heading = document.querySelector('h1');
heading.textContent = 'Tiêu đề mới';
heading.innerText = 'Tiêu đề mới';
```

## `innerHTML` và `outerHTML`

### `innerHTML`

```js
const div = document.getElementById('container');

// Getter: lấy toàn bộ nội dung HTML bên trong element
console.log(div.innerHTML); // "<p>Hello <strong>World</strong></p>"

// Setter: thêm / ghi đè nội dung HTML bên trong
div.innerHTML = '<h2>Tiêu đề mới</h2><p>Nội dung mới</p>';
```

- Lấy **toàn bộ nội dung bên trong** element, bao gồm cả thẻ HTML con.
- Khi dùng làm **setter**, nó sẽ **parse chuỗi HTML và thêm vào bên trong** element đó.

---

### `outerHTML`

```js
const div = document.getElementById('container');

// Getter: lấy element + toàn bộ nội dung bên trong
console.log(div.outerHTML); // "<div id="container"><p>Hello</p></div>"

// Setter: ghi đè chính element đó (thay thế hoàn toàn)
div.outerHTML = '<section><p>Thay thế hoàn toàn</p></section>';
```

| Thuộc tính  | Getter                                          | Setter                                      |
|-------------|------------------------------------------------|---------------------------------------------|
| `innerHTML` | Nội dung **bên trong** element (không gồm chính nó) | Ghi đè nội dung **bên trong** element       |
| `outerHTML` | Nội dung **bên trong + chính element đó**       | Ghi đè **toàn bộ** kể cả chính element đó  |

## Tổng kết

```
document
├── getElementById('id')          → Element
├── getElementsByClassName('cls') → HTMLCollection (live)
├── getElementsByTagName('tag')   → HTMLCollection (live)
├── querySelector('selector')     → Element (first match)
└── querySelectorAll('selector')  → NodeList (static)

element
├── .attribute = value            → Gán attribute có sẵn
├── .className = 'cls'            → Gán class
├── .setAttribute(name, value)    → Thêm attribute bất kỳ
├── .getAttribute(name)           → Lấy giá trị attribute
├── .attributes                   → Danh sách tất cả attribute
├── .innerText                    → Text hiển thị
├── .textContent                  → Toàn bộ text
├── .innerHTML                    → HTML bên trong
└── .outerHTML                    → HTML bao gồm chính element
```
